Para poder ejecutar el proyecto deberas de hacer lo siguiente:

1) crea una **carpeta nueva** 
2) abre la carpeta y click derecho en **abrir terminal** 
3) colocas el siguiente comando 

- git clone https://github.com/JovanyMolina/pruebo-tecnica.git
- cd pruebo-tecnica 
- cd my-app
- y ahi debera de aparecer los archivos 



- abres visual studio code y abres abre la carpeta de my-app

- y en la terminal del vs code, deberas de colocar los siguietes comandos 

- npm install

- npm run dev

- y entras en el link que te da la terminal
